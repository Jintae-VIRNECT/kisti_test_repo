RELEASE

New Features
- 앱 등록 API 추가
- 최신 앱 정보 조회 API 추가
- 앱 인증키 정보 등록 API 추가
- 앱 정보 수정 api 추가 (활성화 or 비활성화, 강제 업데이트 여부)
- 앱 정보 목록 조회 api 추가
- PF-Config 서버 연동, 환경 설정 정보 동기화 기능 추가

Bug Fixes
- 없음

Limitation
- 앱 등록은 apk 파일만 가능
- 최신 앱 정보 조회의 경우, 패키지명 기반으로 단일 앱 정보만 조회 가능
- 앱 인증키 정보 등록의 경우, 인증키 정보가 등록되지 않은 앱 정보만 등록 가능
